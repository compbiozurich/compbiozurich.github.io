### Morning (on site, 9 am)

We will have a lecture and run a set of exercises (on site).

* Overview of the standard genomics data formats
   - FASTA
   - FASTQ
   - SAM
   - BED
   - GFF
   - VCF
* Basic file processing for bioinformatics
   - wc, grep, awk
* Exercises
   - Project
